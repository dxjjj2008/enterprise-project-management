#### 图标库
```

```


## 6. 响应式设计

### 6.1 断点定义
```
┌─────────────────────────────────────────────────────┐
│  xs:    < 576px      │  手机（简化布局）              │
│  sm:   576px - 768px │  大手机 / 小平板（单列布局）    │
│  md:   768px - 992px │  平板（双列布局）              │
│  lg:   992px - 1200px│  小桌面（标准布局）            │
│  xl:   1200px - 1600px│  桌面（完整布局）              │
│  xxl:  > 1600px      │  大屏（宽松布局）              │
└─────────────────────────────────────────────────────┘
```

### 6.2 PC 为主适配策略
```
默认: xl 断点设计
lg: 侧边栏收起、表格简化
md: 侧边栏隐藏（汉堡菜单）、表格卡片化
sm 以下: 全屏抽屉导航、单列布局
```

### 6.3 适配优先级
```
P0: 表格、列表、卡片（手机单列）
P1: 导航、筛选器（汉堡菜单）
P2: 看板、甘特图（建议 PC 使用）
P3: 复杂表单、对比视图（仅 PC）
```
